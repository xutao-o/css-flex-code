# CSS Flex
快速生成CSS Flex布局代码的VS Code插件（类似于Google开发者工具的Flex布局插件）

- 支持多类型的css文件
- 兼容深色与浅色主题
- 自动替换同属性代码
  
## 💻示例

![CSS Flex示例](/images/readme/demo.gif)

## ⚠️注意事项
不建议在压缩后的CSS代码或HTML的行内样式中使用
